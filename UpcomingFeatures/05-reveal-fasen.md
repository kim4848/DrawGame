# Reveal-fasen

| Feature | Beskrivelse | Sværhedsgrad |
|---------|-------------|--------------|
| **Animeret reveal** | Kæden vises trin-for-trin, men uden animation mellem trin (bare fade). Slideshow-effekt ville være sjovere. | Mellem |
| **Zoom på tegninger** | Kan ikke forstørre tegninger i reveal. | Lav |
| **Side-by-side sammenligning** | Vis original ord vs. slutgæt direkte for at se "driften". | Lav |
| **Automatisk afspilning** | Ingen autoplay-tilstand — host skal klikke "Næste" manuelt. Godt til storskærm. | Lav |
| **Download alle kæder** | Gem hele spillets resultat som PDF/billeder. | Mellem |
