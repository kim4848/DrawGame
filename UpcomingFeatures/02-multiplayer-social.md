# Multiplayer & social

| Feature | Beskrivelse | Sværhedsgrad |
|---------|-------------|--------------|
| **Kick spiller (host)** | Host kan ikke fjerne spillere fra lobby. | Lav |
| **Emoji-reaktioner under reveal** | Spillere kan ikke reagere på sjove kæder i realtid. | Mellem |
| **Del resultat som billede** | Eksportér en hel kæde som ét stort billede til deling på sociale medier. | Mellem |
| **Tilskuertilstand** | Nævnt i spec som "out of scope v1", men naturlig næste feature. | Høj |
| **Max antal spillere** | Ingen grænse — 50 spillere ville give 50 runder, hvilket er upraktisk. | Lav |
