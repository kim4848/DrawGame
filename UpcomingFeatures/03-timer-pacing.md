# Timer & pacing

| Feature | Beskrivelse | Sværhedsgrad |
|---------|-------------|--------------|
| **Konfigurerbar timer** | Spec siger hardcoded 90s/30s — host burde kunne vælge (kort/normal/lang). | Lav |
| **"Alle klar" hurtig-skip** | Når alle har indsendt tidligt, venter man til polling opdaterer. Kunne skippe med det samme. | Lav |
| **Pause-funktion** | Ingen måde at sætte spillet på pause (f.eks. hvis nogen skal på toilettet). | Mellem |
