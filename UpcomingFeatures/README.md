## Top 5 prioriteter

1. **Undo/Redo på tegning** — Største UX-frustration lige nu
2. **"Spil igen" med samme gruppe** — Naturligt flow, mangler tydeligt
3. **Fejlnotifikationer (toasts)** — Brugere ved ikke når noget fejler
4. **Konfigurerbar timer** — Lille indsats, stor fleksibilitet
5. **Del kæde som billede** — Det sociale element der gør spillet viralt
